# SSUET AI Assistant

An AI-inspired university chatbot developed for SSUET.

## Features
- Student support chatbot
- Admission guidance
- Fee information
- Department details
- University event updates
- Contact and helpline support

## Technologies Used
- HTML
- CSS
- JavaScript
- Node.js
- Express.js
